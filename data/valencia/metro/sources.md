## Naming convention
L2 station names use both formal Spanish names (Negra Hipólita, Josefa Camejo, etc.)
and sector-based names (Los Sauces, El Viñedo, etc.) interchangeably. OSM tags use the
sector names; this dataset uses the formal names as `name` and sector names as
`alt_names`.
