# Sources — Sistema Teleférico Mukumbarí

- [Wikipedia EN — Mérida cable car](https://en.wikipedia.org/wiki/M%C3%A9rida_cable_car) — station list, altitudes, history.
- [Wikipedia ES — Teleférico de Mérida](https://es.wikipedia.org/wiki/Telef%C3%A9rico_de_M%C3%A9rida) — Spanish-language confirmation.
- [OpenStreetMap](https://www.openstreetmap.org/) via Overpass — **station coordinates** (all 5 stations, last edited 2025-02-12). © OpenStreetMap contributors, ODbL.

## Earlier interpolated coordinates were incorrect
An earlier version of this dataset used linearly-interpolated coordinates along
a 12.5-km axis. Cross-checking against OpenStreetMap (where all 4 named stations
plus an unnamed node — Pico Espejo — are tagged with `network = Sistema
Teleférico de Mérida Mukumbarí`) showed displacements of up to 4.7 km from the
true position (the cable car ascends a winding mountain route, not a straight
line). All 5 coordinates have been replaced with OSM values.
